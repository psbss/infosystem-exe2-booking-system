# Booking_system
> 遺伝的アルゴリズム(GA)を利用した予約システム

## Get Started
```
git clone https://github.com/psbss/infosystem-exe2-booking-system
cd infosystem-exe2-booking-system
```

- Build
```
cd src/
make all
```

- Clean .o file
```
make clean
```

- Clean .a .o file
```
make fclean
```

- Delete generated files / ReBuild
```
make re
```

## Language
- C言語  
メイン利用言語  
プログラム/アルゴリズムの作成に使用
  
- pu  
UML作成に利用
