# Booking_system
> ジョンソン法を利用した予約システム

## Get Started
```
git clone https://github.com/psbss/infosystem-exe2-booking-system
cd johnson-booking-system
```

- Build
```
make
```

- Clean .o file
```
make clean
```

- Clean .a / .o / exe file
```
make fclean
```

- Delete generated files / ReBuild
```
make re
```

## Language
- C言語  
メイン利用言語  
プログラム/アルゴリズムの作成に使用
  
- pu言語
UML作成に利用
